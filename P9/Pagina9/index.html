Tabla Hash

( Hash Table )

Que es una tabla hash

Conoce mas sobre ella

El hasing es una estructura de datos importante que está diseñada para utilizar una función especial llamada función hash, lo cual se usa para asociar un valor dado con una clase específica, permitiendo un acceso más rápido a los elementos. La eficiencia de esta asociación depende directamente de la eficiencia de la función hash utilizada.

Ejemplo en codigo 

import java.util.*;

public class TablaHash {

    public static void main(String[] args) {

        HashMap<String, Integer> edades = new HashMap<>();

        edades.put("Santiago", 19);
        edades.put("Lizeth", 19);
        edades.put("Leslie", 20);

        System.out.println("Tabla Hash: " + edades);

        System.out.println("Edad de Lizeth: " + edades.get("Lizeth"));

        edades.remove("Santiago");

        System.out.println("Después de eliminar: " + edades);
    }
}
