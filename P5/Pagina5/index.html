Listas Enlazadas

Elementos conectados mediante punteros......

██▓▒░⡷⠂Que es una lista enlazada ?⠐⢾░▒▓██

1

Al igual que los arreglos la lista enlazada (linked/list) es una estructura de datos lineal, a diferencia de los arreglos los elementos de una lista enlazada no se almacenan en ubicaciones contiguas de memoria; los elementos están conectados entre sí mediante punteros.

2

Una lista enlazada es una estructura de datos lineal y dinámica, compuesta por nodos secuenciales que no requieren ubicaciones contiguas de memoria.

3

. Cada nodo contiene datos y un puntero (referencia) al siguiente elemento, permitiendo inserciones y eliminaciones eficientes sin reorganizar toda la estructura. El primer nodo se denomina cabeza (HEAD) y el último apunta a NULL.

Ejemplos

██▓▒░⡷⠂EJEMPLO EN CODIGO⠐⢾░▒▓██

class Nodo {
    int dato;
    Nodo siguiente;

    Nodo(int dato) {
        this.dato = dato;
        this.siguiente = null;
    }
}

public class ListaEnlazada {

    Nodo cabeza;

    void agregar(int dato) {
        Nodo nuevo = new Nodo(dato);

        if (cabeza == null) {
            cabeza = nuevo;
        } else {
            Nodo actual = cabeza;
            while (actual.siguiente != null) {
                actual = actual.siguiente;
            }
            actual.siguiente = nuevo;
        }
    }

    void mostrar() {
        Nodo actual = cabeza;

        while (actual != null) {
            System.out.println("Nodo: " + actual.dato);
            actual = actual.siguiente;
        }
    }

    public static void main(String[] args) {
        ListaEnlazada lista = new ListaEnlazada();

        lista.agregar(10);
        lista.agregar(20);
        lista.agregar(30);

        lista.mostrar();
    }
}
