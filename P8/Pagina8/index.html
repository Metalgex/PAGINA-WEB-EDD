██▓▒░⡷⠂Árbol Binario⠐⢾░▒▓██

A diferencia de los arreglos, listas enlazadas, pilas y colas, que son estructuras de datos lineales, los árboles son estructuras de datos jerárquicas. Un árbol binario es una estructura de datos en forma de árbol en el cual cada nodo tiene como máximo 2 hijos, a los que se le denomina hijo izquierdo e hijo derecho se implementa principalmente utilizando enlaces (links) Un árbol binario se representa mediante un puntero al nodo más alto del árbol. Si el árbol está vacío entonces el valor de la raíz es NULL.

2

██▓▒░⡷⠂Árbol de Búsqueda Binario⠐⢾░▒▓██

Un árbol de búsqueda binaria es un árbol binario que cumple las siguientes propiedades adicionales. • La parte izquierda del nodo raíz contiene claves menores que la clave del nodo raíz • La parte derecha del nodo raíz contiene claves mayores que la clave del nodo raíz • No existen claves duplicados en el árbol binario Un árbol binario que cumple con estas propiedades se conoce como árbol de búsqueda binaria (BST).

Ejemplos

██▓▒░⡷⠂EJEMPLO EN CODIGO⠐⢾░▒▓██

class Nodo {
    int dato;
    Nodo izquierda, derecha;

    Nodo(int dato) {
        this.dato = dato;
        izquierda = derecha = null;
    }
}

public class ArbolBinario {

    Nodo raiz;

    Nodo insertar(Nodo raiz, int dato) {
        if (raiz == null) {
            raiz = new Nodo(dato);
            return raiz;
        }

        if (dato < raiz.dato)
            raiz.izquierda = insertar(raiz.izquierda, dato);
        else
            raiz.derecha = insertar(raiz.derecha, dato);

        return raiz;
    }

    void inorden(Nodo raiz) {
        if (raiz != null) {
            inorden(raiz.izquierda);
            System.out.println(raiz.dato);
            inorden(raiz.derecha);
        }
    }

    public static void main(String[] args) {
        ArbolBinario arbol = new ArbolBinario();

        arbol.raiz = arbol.insertar(arbol.raiz, 50);
        arbol.insertar(arbol.raiz, 30);
        arbol.insertar(arbol.raiz, 70);
        arbol.insertar(arbol.raiz, 20);
        arbol.insertar(arbol.raiz, 40);

        arbol.inorden(arbol.raiz);
    }
}
